## 🎯 목표

지금은 손절 기준이 **마지막 매수가 - 2N** (고정값) 입니다.
이것을 **매수 후 최고가 - 2N** (추적값) 으로 바꿉니다.

가격이 오를수록 손절가도 따라 올라가서, 많이 오른 종목이 갑자기 급락할 때 수익을 지켜줍니다.

핵심 결정사항:
- 최고가는 10분 주기 실행마다 현재가로 갱신 (실시간)
- 피라미딩(추가 매수)해도 최고가는 초기화하지 않고 전체 보유 기간 최고가 유지
- 기존 손절(마지막 매수가 - 2N)과 병행 — 둘 중 **높은 쪽**이 유효 손절가

변경 위치: 411줄 (`ma5 = indicators.get(...)`) 이후에 블록 추가  
변경 내용: ① 최고가 갱신 ② 트레일링 손절가 계산 ③ `stop_loss_price_val` 재취득

```python
# ─────────── 추가 블록 (ma5 읽은 직후) ───────────
# 매수 후 최고가 갱신 및 트레일링 손절가 상향 처리
atr_n            = indicators.get("atr", 0.0)
high_since_entry = pos.get("high_since_entry", pos.get("last_buy_price", 0))
_updated         = False

# 현재가가 최고가를 넘으면 최고가 갱신
if high_since_entry > 0 and current_price > high_since_entry:
    pos["high_since_entry"] = current_price
    high_since_entry        = current_price
    _updated                = True

# 최고가 - 2N이 기존 손절가보다 높으면 손절가 상향
if atr_n > 0 and high_since_entry > 0:
    trailing_stop = int(high_since_entry - 2 * atr_n)
    if trailing_stop > pos.get("stop_loss_price", 0):
        pos["stop_loss_price"] = trailing_stop
        _updated               = True

if _updated:
    save_position_state(position_state)

# 트레일링으로 손절가가 올라갔을 수 있으므로 최신값으로 재취득
stop_loss_price_val = pos.get("stop_loss_price", 0)
# ─────────────────────────────────────────────────
```
