# ls-ask: 전략 질문 답변 커맨드

사용자의 질문에 **답변만** 한다. 코드는 절대 수정하지 않는다.

---

## 규칙

1. **코드를 수정하지 않는다.** 파일 편집, 생성, 삭제 일체 금지.
2. **질문에 대한 답변만 한다.** 불필요한 설명, 제안, 다음 단계 안내 없이 묻는 것만 답한다.
3. **짧고 명확하게** 답한다.
4. 모르는 것은 모른다고 한다.

---

## 사용법

```
/ls-ask [질문]
```

예:
```
/ls-ask 매수 조건이 뭐야?
/ls-ask 손절 기준이 어떻게 돼?
/ls-ask ATR이 뭐야?
```

---

## 답변 방식

- 전략 관련 질문 → `ls_hybrid_turtle.md`, `CLAUDE.md` 기준으로 답변
- 코드 동작 관련 질문 → 해당 파일을 읽어서 답변
- 어떤 경우에도 코드 수정 없이 답변만 한다
